# Cartera — Sistema de Cartas · Cobros Venta Directa (Proyecto CARD)

Prototipo funcional del CRM de gestión y generación de cartas de cobranza para Grupo PDC.
Acceso: modo pruebas (sin claves). En producción: gestores con `Avon`+código de empleado.

**Estado:** prototipo con datos de demostración. Pendiente de conexión a Supabase y textos legales definitivos.
